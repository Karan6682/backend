# How to Get Permanent Meta Access Token (Step-by-Step Script)

This guide explains how to generate a permanent System User Token for your CloudCRM.

---

### Step 1: Open Business Settings
1. Go to your browser.
2. Type and open: `business.facebook.com/settings`
3. Select your **Business Account** if asked.

---

### Step 2: Creates a System User
1. On the left sidebar menu, look for **Users**.
2. Click on **System Users**.
3. Use the **Add** button.
   - **System User Name:** Type `CRM Admin`
   - **System User Role:** Select `Admin`
4. Click **Create System User**.

---

### Step 3: Add Assets (Important!)
1. Select the new user `CRM Admin` you just made.
2. Click the **Add Assets** button.
3. Select **Apps** from the menu.
4. Select your WhatsApp App Name (e.g., `Pathostar`).
5. Turn on the toggle for **Manage App (Full Control)**.
6. Click **Save Changes**.

---

### Step 4: Generate Token
1. Click the **Generate New Token** button.
2. Select your **App** from the dropdown.
3. **Token Expiration:** Select `Permanent` (or Never).
4. **Scrollo down to Permissions (Scopes):**
   * Search and check: `whatsapp_business_management`
   * Search and check: `whatsapp_business_messaging`
5. Click **Generate Token**.

---

### Step 5: Copy & Connect
1. A popup will show your Access Token.
2. Click to **Copy** it immediately (you won't see it again!).
3. Go to your **CloudCRM Dashboard**.
4. Paste it in the **Meta Access Token** box.
5. Click **START DISCOVERY**.

---

**(Closing)**
"That's it! Your CloudCRM is now permanently connected effectively. Happy Messaging!"
